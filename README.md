# NIRD Quest

Plateforme gamifiée pour promouvoir le **Numérique Inclusif, Responsable et Durable** à travers des quêtes, missions et contenus pédagogiques.

## 🎯 Objectif

Accompagner la communauté NIRD à travers :
- **Mobilisation** : sensibilisation aux enjeux
- **Expérimentation** : premières actions (Linux, réemploi)
- **Intégration** : adoption durable des pratiques

## 🚀 Démarrage

```bash
# Installation
npm install

# Développement
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000)

## 📂 Structure

- `/app/page.tsx` - Page d'accueil
- `/app/quests` - Système de quêtes gamifiées
- `/app/missions` - Missions communautaires
- `/app/learn` - Ressources pédagogiques
- `/app/data/nirdData.ts` - Données des quêtes et contenus

## 🎮 Fonctionnalités

- **XP et niveaux** : progression gamifiée (120/240/400 XP)
- **Quêtes** : défis par pilier (Inclusif, Responsable, Durable)
- **Missions** : contributions pour la communauté
- **Animation d'intro** : séquence NIRD au chargement
- **Easter egg** : Machine de Rube Goldberg cachée et Snake

## 🛠️ Technologies

- **NodeJS 25.2.1**
- **Next.js 15** (App Router)
- **TypeScript**
- **CSS Modules**
